API de Productos

Sistema REST para gestión de productos con Django REST Framework

Django 5 DRF PostgreSQL Render

Rendimiento

Respuestas ultra rápidas con Django optimizado

Seguridad

Autenticación y validación integrada

Cloud

Desplegado en Render con PostgreSQL

API REST

Endpoints RESTful para integración

Endpoints Disponibles

GET
/api/productos/

Obtiene la lista completa de productos

POST
/api/productos/

Crea un nuevo producto en el sistema

GET
/api/productos/{id}/

Obtiene un producto específico por ID

PUT
/api/productos/{id}/

Actualiza completamente un producto

DELETE
/api/productos/{id}/

Elimina un producto del sistema

Panel de Administración

GET
/admin/

Acceso al panel de administración de Django (requiere login)

Probar API Ahora Panel Admin
Ejemplo de Uso

# Crear un producto con curl

curl -X POST https://eva3-backend.onrender.com/api/productos/ \

-H "Content-Type: application/json" \

-d '{

"nombre": "Laptop Gamer",

"descripcion": "Laptop para gaming",

"precio": 1299.99,

"stock": 15

}'
